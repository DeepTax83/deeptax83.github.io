프로필

통계적 사고와 엔지니어링 실행력을 결합해, 비정형 문제를 데이터 기반 제품으로 전환하는 실전형 빌더입니다. 요구사항 정의 → 데이터 파이프라인 설계 → LLM·규칙 하이브리드 → 대시보드/웹 UI 배포까지 끝단 납품을 목표로 합니다. 빅데이터분석기사 대비 학습을 바탕으로 Pandas·시각화·실험설계에 강점이 있으며, 자동화에서 재현성·표준화를 우선합니다.

최근 Job Buddy로 스펙 파싱–JD 태깅–서류 템플릿 엔진–인터뷰 Q/A 파이프라인을 구성해 서류 초안 생성 시간 약 80% 절감을 달성했고, mindtrip에선 예산·동선 제약을 반영한 일정 최적화와 지도 시각화를 구현했습니다. 제 작업은 문제 정의의 선명도, 간결한 데이터 모델, 사용자 체감 가치를 양 축으로 합니다.

고객 대면(영업·컨설팅) 경험을 바탕으로 요구사항을 데이터 문제로 재정의하고, 가설–실험–검증 루프를 빠르게 돌립니다. 현재는 Job Buddy·mindtrip의 제품 완성도와 안정적 배포/운영(로깅·모니터링·재현성)에 집중하고 있습니다.

  • 스택 · Python, Pandas, NumPy, Matplotlib, SQL(SQLite/MySQL), PyTorch, TensorFlow(GPU), Git/GitHub

대표 프로젝트

Job Buddy — 구직 자동화 어시스턴트

Python · LLM · RAG 효율 +80%

사용자 스펙 입력 → 기업 추천 → 이력서/커버레터 자동 작성 → 인터뷰 가이드 → 최종 합격까지 파이프라인화.

  • 핵심: 스펙 파서, JD 태깅, 템플릿 엔진, 면접 Q/A 생성
  • 성과: 샘플 프로필 기준 지원서 초안 생성 시간 80% 절감

mindtrip — AI 여행 플래너

Python · Maps · Prompt 브랜딩

취향/예산 기반 일정·예산·지도 시각화 자동 생성. 화이트 래빗 브랜드 스토리 구축.

  • 핵심: 일정 최적화, 예산 산출, 경로 시각화
  • 브랜딩: "히위고~!" 슬로건, 토끼 로고/프롬프트 셋

부동산 실거래가 분석 & 지도 서비스

Pandas · Folium · FastAPI · Streamlit 데이터 제품

국토부 실거래가를 정제·집계해 지역/기간별 단가·거래량을 시각화하고, 대시보드/간단 API까지 연결.

  • 데이터: CSV ETL → SQLite 적재, 파생지표(㎡당 단가, 분기)
  • API: /stats?region=…&from=… 평균·중위·거래량 JSON
  • 지도/대시보드: Choropleth + 필터/차트
